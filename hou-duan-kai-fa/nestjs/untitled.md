### 主要文件说明

| `app.controller.ts` | 具有单一路由的基本控制器。 |
| :--- | :--- |
| `app.controller.spec.ts` | 控制器的单元测试。 |
| `app.module.ts` | 应用程序的根模块。 |
| `app.service.ts` | 具有单一方法的基本服务。 |
| `main.ts` | 使用核心功能`NestFactory`创建Nest 应用程序实例的应用程序的入口文件。 |
