# 创建项目

新建目录，执行`bee new bee_go_project`创建一个项目

```bash
# 进入目录
cd StudyWork/Go/
# 创建文件夹来放置beego项目
mkdir beeGo
cd beeGo
# 初始化项目
bee new bee_go_project
```

目录结构

```text
quickstart
|-- conf
|   `-- app.conf
|-- controllers
|   `-- default.go
|-- main.go        入口文件
|-- models
|-- routers
|   `-- router.go
|-- static
|   |-- css
|   |-- img
|   `-- js
|-- tests
|   `-- default_test.go
`-- views
    `-- index.tpl
```
