# 数组

数组是一段固定长度的连续内存区域。

在Go语言中，数组从声明时就确定，使用时可以修改数组成员，但是数组大小不可变化。

```go
//数组写法
// var 数组变量名 [长度]类型

var arr [6]int

//初始化数组

var arr1 = [3]int{1,2,3}
arr2 := [3]int{4,5,6}

//编译器确定数组
arr3 :=[]int{7,8,9,0}
```

数组遍历

```go
sArr := [6]int{6, 6, 7, 7, 8, 8}
for i,v := range sArr{
	fmt.Println(i,v)  
}
```
